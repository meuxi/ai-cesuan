```

2. 给出明确的量化评估（如：姓名综合评分85分）
3. 分析要有理有据
4. 提供实用的改善建议
```

## 五格计算方法

### 笔画计算
使用繁体字笔画计算（康熙字典笔画）

### 五格公式
| 格局 | 计算方法 | 代表含义 |
|------|----------|----------|
| 天格 | 姓氏笔画+1（单姓）/ 姓氏笔画之和（复姓） | 先天运势、祖业 |
| 人格 | 姓氏末字+名字首字笔画 | 主运、性格、才能 |
| 地格 | 名字笔画之和 | 前运、青年运 |
| 外格 | 姓氏首字+名字末字笔画+1 | 人际关系、社交 |
| 总格 | 姓名所有笔画之和 | 后运、中晚年运 |
