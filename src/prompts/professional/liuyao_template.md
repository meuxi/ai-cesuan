## 核心断卦十二法则

### 一、月建日辰为纲领
- 月建为纲：月建主宰爻的旺衰根本
- 日辰为领：日辰可生克冲合各爻
- 月日同功同权：月建日辰皆能生克冲合

### 二、旺衰五态（量化评估）
| 状态 | 描述 | 力量 |
|------|------|------|
| 旺 | 当令最旺 | 100% |
| 相 | 受月令所生 | 80% |
| 休 | 生月令者 | 50% |
| 囚 | 被月令所克 | 30% |
| 死 | 克月令者 | 10% |

### 三、暗动与日破
- 暗动：静爻旺相逢日冲，虽静犹动
- 日破：静爻休囚逢日冲，冲而破散

### 四、空亡体系
- 真空：静爻空亡，无力作用
- 动空不空：动爻虽空，待填实可用
- 冲空不空：日辰冲空，空而不空
- 临月建不空：临月建之爻不受空亡

### 五、用神为核心
| 测事类型 | 用神取法 |
|----------|----------|
| 财运 | 妻财爻 |
| 事业 | 官鬼爻 |
| 考试 | 父母爻 |
| 婚姻 | 妻财/官鬼 |
| 健康 | 用神或世爻 |
| 子女 | 子孙爻 |

### 六、原神忌神仇神
- 原神：生用神者
- 忌神：克用神者
- 仇神：克原神者
